
李高峰-牛客网-牛人计划-初级项目课

图片分享网站项目

开发环境及工具：
		pyhton 2.7.8 
		PyCharm
		Flask (0.11.1)
		Flask-Login (0.3.2)
		Flask-Mail (0.9.1)
		Flask-Script (2.0.5)
		Flask-SQLAlchemy (2.1)
		Sqlite
		Jinja2 (2.8)
		qiniu (7.0.7)
		requests (2.10.0)
		SQLAlchemy (1.0.13)

基本功能：
		首页图片及评论展示及更多
		个人主页图片展示、上传及更多
		详情页图片评论展示，评论添加
		注册、登陆页面
		
扩展功能：
		邮箱注册和邮箱激活，激活链接加密
		未激活态的评论和上传限制
		未登陆用户的访问他人主页控制、评论控制
		邮箱或用户名任选登陆
		登陆后增加登出图标和登出入口
		图片访问pv记录
		后台管理员删图和删评论，置顶/取消置顶图片
		首页图片展示排序顺序为：置顶值>pv>created_date
		从unsplash和tuweng网站分别爬图和评论来填充数据库
		
未实现功能：
		首页评论添加
		首页及详情页点赞的实现
		首页滚动到底部自动加载更多
